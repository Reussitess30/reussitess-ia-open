# Validation des Liens Amazon - Documentation

## Format des Liens d'Affiliation Amazon

### Liens Boutiques Personnelles
Format: `https://www.amazon.{domain}/shop/{username}`

Exemples valides :
- https://www.amazon.com/shop/amourguadeloupe
- https://www.amazon.fr/shop/amourguadeloupe
- https://www.amazon.de/shop/amourguadeloupe

### Liens Boutiques Influenceurs
Format: `https://www.amazon.{domain}/shop/influencer-{id}`

Exemples valides :
- https://www.amazon.com/shop/influencer-fb942837
- https://www.amazon.fr/shop/influencer-fb942837

## Vérification des Liens

Tous les liens dans notre application suivent le format correct et pointent vers :
1. Boutiques personnelles Amazon vérifiées
2. Programmes d'affiliation Amazon légitimes
3. Domaines Amazon officiels uniquement

## Conformité

✅ Tous les liens respectent les Conditions d'Utilisation du Programme Partenaires Amazon
✅ Disclaimer d'affiliation visible sur toutes les pages
✅ Aucun lien de tracking tiers non autorisé
✅ Pas de redirection cachée ou de cloaking

## Mise à Jour

Les liens sont vérifiés et mis à jour régulièrement pour assurer :
- La validité des boutiques
- Le respect des nouvelles politiques Amazon
- L'optimisation des conversions
